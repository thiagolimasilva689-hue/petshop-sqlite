
# 🐾 Pet Shop - Banco de Dados SQLite

Projeto de banco de dados para um pet shop usando Python + SQLite.

## 🛠️ Funcionalidades
- Criar banco e tabela
- Inserir animais
- Consultar com filtros (vacinados, peso, espécie, idade)
- Atualizar dados
- Remover registros
- Contar vacinados e calcular peso médio

## 🚀 Tecnologias
- Python 3
- SQLite3

## 📁 Arquivos
- `petshop.py` — Código principal
- `petshop.db` — Banco de dados
